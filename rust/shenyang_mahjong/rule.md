# 沈阳麻将标准规则

## 一、普通牌型胡牌硬性前提

- 三门齐：万、筒、条三种花色必须都有，缺任意一门不能胡
- 有幺九：手牌必须包含 1/9 序数牌，或东南西北中发白任意字牌（字牌算幺九）
- 开门：普通平胡必须开门；飘胡通常必须开门，但全横且以中 / 发 / 白作掌时可闭门胡；七对、清一色可闭门胡牌
- 有横：必须有一组三张相同的刻子；若无刻子，一对中 / 发 / 白当将可替代
- 吃牌：不存在全局禁吃；默认允许闭门玩家用第一次吃开门。`allow_first_chi=0` 时闭门玩家不能用吃开门，allow_first_chi = 0 时允许中发白做掌的无横闭门胡

## 二、核心可胡牌型与基础番数

- 平胡（屁胡） 1 番 顺子 + 刻子 + 将的普通牌型，不能断幺,
- 飘胡（对对胡） 3 番 4 组刻子 / 杠 + 1 对将；全程不能吃，必须三门齐、有幺九，缺门直接不成立；通常需要碰 / 明杠开门，但全横且以中 / 发 / 白作掌时允许闭门胡
- 七小对 固定 4 番 7 个对子组成；无需三门齐、无需开门(也不可能开门)、无需幺九，不受普通胡牌条件约束
- 清一色 固定 4 番 纯万 / 纯筒 / 纯条，不能带任何字牌；无需三门齐

## 三、通用加番项（所有牌型可叠加）

#### 1. 杠牌类

- 普通明杠：+1 番
- 普通暗杠：+2 番
- 中发白碰（横）：+1 番
- 中发白明杠：+2 番
- 中发白暗杠：+4 番

#### 2. 胡牌方式类

- 杠开（杠上开花）：+1 番，开杠补牌直接胡牌
- 抢杠胡：+1 番，胡他人开明杠的那张牌
- 海底捞月：+1 番，牌局最后分张阶段自摸胡牌

#### 3. 状态类

- 坐庄：+1 番（庄家胡牌、输牌均加番）
- 闭门：输家未开门，每家 + 1 番；三家全没开门为「三家闭」，+2 番
- 和单张：夹张、边张、单钓、单胡幺，均 + 1 番
- 手把一：+1 番，仅限飘胡，最后只剩 1 张牌单钓将
- 四归一：+1 番，四张相同牌未开杠、完整在手牌里（包括碰，吃，在明牌的）
- 喜杠: 发完牌后，非庄家摸完第一章牌后，手里有中发白，或者东南西北，这些牌叫做喜杠，加一番，东南西北需要摸一张牌
