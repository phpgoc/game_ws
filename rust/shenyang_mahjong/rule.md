# 沈阳麻将标准规则

## 一、普通牌型胡牌硬性前提

- 三门齐：万、筒、条三种花色必须都有，缺任意一门不能胡
- 有幺九：手牌与副露必须包含 1/9 序数牌，或东南西北中发白任意字牌（字牌算幺九）；已声明喜杠中的字牌也满足此项
- 开门：普通平胡必须开门；飘胡通常必须开门，但全横且以中 / 发 / 白作掌时可闭门胡；七对、清一色可闭门胡牌
- 有横：必须有一组三张相同的刻子；若无刻子，一对中 / 发 / 白当将可替代；已声明的喜杠也可提供横，但仍不算开门
- 吃牌：不存在全局禁吃；默认允许闭门玩家用第一次吃开门。`allow_first_chi=0` 时闭门玩家不能用吃开门，但通过碰 / 明杠开门后仍可吃；同时允许其余牌全部组成顺子、以中 / 发 / 白作掌的无横平胡闭门胡，已声明的喜杠不阻断这个闭门顺子例外。该例外与上文全横闭门飘胡是两条独立规则。任何已经发生的吃都算开门

## 二、核心可胡牌型与基础番数

- 平胡（屁胡） 1 番 顺子 + 刻子 + 将的普通牌型，不能断幺,
- 飘胡（对对胡） 3 番 4 组刻子 / 杠 + 1 对将；全程不能吃，必须三门齐、有幺九，缺门直接不成立；通常需要碰 / 明杠开门，但全横且以中 / 发 / 白作掌时允许闭门胡
- 七小对 固定 4 番 7 个对子组成；无需三门齐、无需开门(也不可能开门)、无需幺九，不受普通胡牌条件约束
- 清一色 固定 4 番 纯万 / 纯筒 / 纯条，不能带任何字牌；无需三门齐

## 三、通用加番项（所有牌型可叠加）

#### 1. 杠牌类

- 普通明杠：+1 番
- 普通暗杠：+2 番
- 中发白横（明碰或手牌中的暗刻）：+1 番；四张未开杠时仍按暗刻计此项，并可另叠加四归一
- 中发白明杠：+2 番
- 中发白暗杠：+4 番

#### 2. 胡牌方式类

- 杠开（杠上开花）：+1 番，开杠补牌直接胡牌
- 抢杠胡：+1 番，胡他人开明杠的那张牌
- 海底捞月：+1 番，牌局最后分张阶段自摸胡牌
- 普通自摸不额外加番，只改变付款方式；仅在同时满足杠开、海底等明确加番条件时叠加对应番数

#### 3. 状态类

- 坐庄：+1 番（庄家胡牌、输牌均加番）
- 闭门：输家未开门，每家 + 1 番；三家全没开门为「三家闭」，+2 番
- 和单张：夹张、边张、单钓、单胡幺，均 + 1 番
- 手把一：+1 番，仅限飘胡，最后只剩 1 张牌单钓将
- 四归一：+1 番，四张相同牌未开普通杠、完整分布在手牌和明牌里（包括碰、吃、喜杠中公开的牌）
- 喜杠：闲家第一次正常摸牌后可主动声明；中、发、白各一张组成一组喜杠，东、南、西、北各一张组成另一组喜杠，声明后移出手牌并公开，每组 +1 番，两组同时成立累计 +2 番。中发白喜杠不补牌，东南西北喜杠补摸一张。喜杠不算开门，喜杠补牌胡也不算杠开；声明窗口在该闲家首次弃牌后关闭

## 四、番数结算与封顶

- 每家付款先累计该赢家牌型番、庄家番和该付款人的闭门番，再按 `2^番数` 换算分数：0 番 1 分、1 番 2 分、2 番 4 分，依次翻倍
- 自摸时三家分别按各自最终番数付款；点炮时只由点炮者向每名赢家付款
- 点炮允许一炮多响，不存在截胡；同一张牌有多名玩家合法确认胡时全部作为赢家同时结算，抢杠胡同样处理
- `max_fan` 是兼容旧接口保留的设置名，值表示每家单次付款的封顶分数，不是番数；例如封顶 50 时，5 番付 32 分，6 番及以上均付 50 分
- 封顶在庄家番、闭门番等全部累计并换算成分数后，对每家付款分别生效

## 五、听牌声明

- 真人玩家发现本次弃牌后至少有一种牌可满足完整沈阳麻将胡牌规则时，可以在弃牌时自愿声明听牌；内置 AI 不主动声明听牌
- 声明后每次只能弃掉刚摸到的牌；不能再吃、碰、普通杠或喜杠，响应他人出牌时只保留合法的胡牌操作
- `ting_fan=0` 时声明听牌不加番，默认使用该值；`ting_fan=1` 时合法胡牌额外 +1 番，并在换算分数和封顶前累计
- 听牌声明不放松三门齐、有幺九、开门、有横及特殊闭门例外等胡牌条件；无效牌型不能靠听牌番恢复为合法胡牌
