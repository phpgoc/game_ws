# 一、核心决策总纲
- 飘胡是常态最优目标，收益稳定、成型难度适中，是绝大多数牌局的首选路线
- 坐庄优先速胡，不硬憋高番，优先保证胡牌率；庄家输赢均加番，小胡也不亏
- 牌型定型后不轻易转路线，避免两头落空、最终丧失胡牌资格
- 普通牌型（平胡 / 飘胡）必须守住三门不缺、留幺九两条底线；平胡通常还要开门，飘胡通常也要开门，但全横且以中 / 发 / 白作掌时可闭门胡；`allow_first_chi=0` 时，另有其余牌全部组成顺子、以中 / 发 / 白作掌的无横平胡闭门例外；七对 / 清一色不受这些普通牌型限制
- 需根据房间单家付款封顶分数动态调整：当前可胡时把牌面番、庄家番和本次各付款人的闭门番合并后，分别按 `2^番数` 换算为分数，再判断是否超过半封顶或达到封顶；接近封顶时停止憋番，优先胡牌落地
- 牌型彻底崩盘（缺门、无幺九、离听牌极远）时，防守优先，不点炮就是赢

# 二、分阶段牌型定型策略

## 1. 前期（牌局进度 0~30%，摸牌前 5~6 张）：定型期

-   闲家 5 对及以上：直接锁定七对路线，全程不碰，弃牌优先打孤张字牌、非幺九孤张；庄家若普通牌型仍可行，不因 5 对硬锁七对，仅在已到 6 对临听或普通牌型因缺门等原因不可行时保留七对路线
-   4 对且缺门：奔七对，缺门状态下飘胡 / 平胡均无法胡牌，七对是唯一合规出路
-   3~4 对且三门齐：优先奔飘胡，能碰就碰，保留所有对子，不拆对
-   3 对以下 + 顺子搭子多：奔平胡，保留门子，优先凑「横」（刻子或中发白做将）
-   单花色 8 张以上：可奔清一色（固定 4 番），直接打掉另外两门孤张
-   硬性红线：普通牌型此阶段不能打光任意一门，必须保留至少 1 张幺九 / 字牌；已明确奔清一色时例外
- 
## 2. 中期（牌局进度 30%~70%）：推进期
-   飘胡路线：优先碰牌推进，保证三门齐全，对子不拆，往「4 刻 + 1 将」靠拢
-   平胡路线：优先凑顺子，同时保横、保幺九；进度过半后优先开门，降低点炮风险
-   七对路线：只摸不碰，留对拆孤，优先留生张对子、字牌对子
-   牌型崩盘：立刻转防守，优先碰牌 / 明杠开门，降低点炮输番成本，哪怕破坏牌型也可
- 
## 3. 后期（牌局进度 70% 以上，剩余牌墙≤20 张）：决胜 / 防守期

-  已听牌：能胡就胡；仅当本次各付款人的最终付款番都恰好差 1 番封顶，且听口胡牌概率极高时才允许过胡，否则不硬憋
-  未听牌：彻底防守，只打绝对安全牌，以不点炮为核心目标

#  三、碰 / 吃 / 杠精细操作规则

## 1. 碰牌规则

-   总原则：奔平胡 / 飘胡时，能碰就碰为默认操作，存在 3 个例外：
-   碰后会导致缺门，且无法补回该门牌时，绝对不碰
-   奔平胡且已有足够横，碰后会破坏优质两面搭子时，可选择不碰
-   纯防守局（已确定胡不了），可放弃碰牌避免暴露牌型
-   奔七对 / 清一色时：一律不碰，哪怕中发白也不碰
-   中发白碰优先级最高：既算横、又算开门、还加 1 番，能碰必碰
-   奔七对和中发白的碰，具体怎么取舍斟酌下，看现在的对子是否缺门，不缺门就提大大提高碰的优先级，还要评估现在的对子再已打牌里是否已经打光，已经打光就不碰了，奔不了飘了。

## 2. 吃牌规则

-  默认允许闭门时用第一次吃开门；`allow_first_chi=0` 时，AI 闭门阶段不吃，已有碰 / 明杠等开门副露后仍可按以下策略吃牌，并把其余牌全部组成顺子、以中 / 发 / 白作掌的无横闭门平胡视为合法路线
-   前期（≤50% 进度）：不吃，靠碰开门，保留牌型灵活性
-   中期（>50% 进度）：仅两种情况吃：① 吃后直接听牌；② 吃后能开门，且自身牌差急需降低点炮番
-  飘胡路线全程不吃，吃牌直接作废飘胡资格
## 3. 杠牌规则

### 明杠

-  牌差需开门：立刻杠，优先完成开门、降低点炮风险
-  牌好奔大番：留到听牌后再杠，博杠开加番
-  当前可见番换算后的分数已超过半封顶，且明杠后可直接达到封顶时，不再延后，立即杠并争取补牌
-  中发白明杠：优先级极高，能杠就杠，番数收益远大于普通杠
- 
### 暗杠
-  暗杠不算开门，需开门的普通牌型，不要为了暗杠耽误开门进度
-  已开门、奔大番时，暗杠优先留到听牌后开，最大化杠开概率
-  当前可见番换算后的分数已超过半封顶，且暗杠后可直接达到封顶时，不再延后，立即杠并争取补牌
-  飘胡暗杠不影响牌型，平胡暗杠也算「横」
-  特殊取舍：四张同牌能凑「四归一」且加番收益≥杠番时，可不开杠，保留四归一

### 喜杠

- 闲家首摸出现东南西北时通常主动声明，获得 +1 番并补摸一张；它不算开门，补牌胡也不算杠开
- 中发白喜杠通常主动声明；仅当中发白中至少两种恰好各有两张、形成多个对子时保留对子，不亮喜杠。三张刻子或四张四归一不按对子计，七对 / 清一色路线也不作为额外拒绝理由
- 两种喜杠同时成立可依次声明并累计 +2 番，但声明后字牌组占用牌型组数，普通胡牌仍要重点检查三门齐，不能因加番走成缺门死牌
- 喜杠中公开的单张仍参与四归一统计；若其余三张组成暗刻，可在喜杠番之外再得四归一 +1 番
- `allow_first_chi=0` 时，喜杠不阻断其余牌全部组成顺子、以中 / 发 / 白作掌的无横闭门平胡路线，AI 的听牌与番数估算仍按该路线计算

# 四、听牌选择与番数优化

## 1. 速胡策略（坐庄 / 牌差 / 牌局后期）

-   优先选听口数量最多的听牌（如两面听、三面听），胡牌率为第一优先级，番数让步。
- 
## 2. 憋番策略（闲家 / 牌好 / 前中期）

- 优先选带加番的听口，番数收益优先，加番听口优先级从高到低：
- 飘胡手把一单钓：同时拿「手把一 + 和单张」双重加番
- 普通单钓将：和单张 + 1 番
- 嵌张（夹张）、边张：和单张 +1 番；例如持 7、8 单等 9 既是边张又是单胡幺，但同属一个“和单张”番项，只加 1 番，不重复累计


## 3. 硬性红线

- 普通牌型听牌后必须满足「三门齐、有幺九、已开门」，不能为了憋番听成无效牌型；闭门全横且以中 / 发 / 白作掌的飘胡始终例外，`allow_first_chi=0` 时，其余牌全部组成顺子、以中 / 发 / 白作掌的无横闭门平胡也例外。

# 五、弃牌与防点炮策略

## 1. 安全牌优先级（从高到低）

- 桌面已打过的熟张字牌（东南西北中发白）
-  桌面打过多张的熟张序数牌
-  手牌孤张风牌
-  中间张熟张（3~7）
-  幺九熟张
-  生张字牌
-  生张幺九
-  生张中间张（最高危）
- 
## 2. 分阶段弃牌逻辑

-  前期：先打孤张字牌、无用幺九，保留门子和对子
-  中期：打无用边张，保留核心搭子，优先跟打桌面熟张
-  后期：绝不打生张，只跟熟张；可通过对手弃牌判断其缺门花色，优先打他的缺门牌
- 
## 3. 防守判断

-  对手碰了 3 组以上：大概率奔飘胡，警惕其将牌花色，少打生张对子
-  对手全程没碰没杠：大概清一色或七对，警惕生张和冷门牌
-  自身牌烂胡不了：先开门降点炮番，再全打安全牌，不点炮就是盈利

# 六、特殊牌型专属策略

## 1. 七小对（固定 4 番）

- 定型阈值：起手 4 对可考虑，5 对及以上锁死路线
- 2 分封顶房间以速胡为先，五对及以下不锁七对；已经六对临听时仍保留七对路线
- 操作要点：全程不碰、不普通杠，拆孤留对，优先留生张对子、字牌对子；喜杠按专节主动声明，声明后退出七对路线
- 听牌选择：单钓优先选生张、风牌、幺九，避免钓中间熟张
- 核心优势：无需三门齐、无需开门、无需幺九，是缺门牌型的唯一合规退路

## 2. 清一色（固定 4 番）

- 定型阈值：起手同花色≥8 张，否则不建议硬做，风险极高
- 2 分封顶房间不追清一色；当前缺门、普通牌型不成立时仍可保留清一色退路
- 操作要点：前期快速打掉另外两门，全程不碰（包括主花色），字牌逐步清理；喜杠按专节主动声明，声明后退出清一色路线
- 规则红线：清一色不能带任何字牌，带字牌不算清一色（标准规则）
- 番数叠加：可叠加杠、庄、杠开、海底等明确加番项；普通自摸只改变付款方式，本身不额外加番

# 七、庄闲差异化策略

## 1. 庄家策略

- 核心目标：速胡优先，番数次之；庄家输赢均加番，平胡也有稳定收益
- 操作：优先开门、早听牌，不硬憋飘胡 / 七对，能平胡就平胡
- 禁忌：不做高风险大牌（清一色、强行七对），避免小牌点大炮
- 
## 2. 闲家策略

- 核心目标：追求番数收益，平胡收益低，可适当奔飘胡 / 七对
- 操作：牌好就憋番，牌差就速胡或防守，避免用小牌硬刚庄家大牌

# 八、憋大番常见招式

- 留单胡口：如 5 张牌 2、2、4、5、7 时，打 4 改听 6 的嵌张，拿和单张加番；坐庄速胡策略下则打 7 听 3、6 两面，优先胡牌率
- 杠牌延后：所有明杠 / 暗杠留到听牌后再开，最大化杠开概率，杠开直接 + 1 番(要评估憋杠不要拆自己牌型)
- 手把一憋番（飘胡专属）：主动碰满 4 组刻子，手里只留 1 张将牌单钓，同时拿「手把一 + 和单张」双重加番
- 四归一留牌：4 张相同的牌不开杠，完整保留在手牌中，拿四归一加番；适合已听牌、开杠会破坏听口的场景
- 边幺留听：主动改听边张、单幺九
- 如果当前番数换算后的分数已经大于封顶分数的一半，所有憋番策略都要停止，优先胡牌落地；例如 5 番为 32 分，封顶 50 时已经超过半封顶，但尚未封顶，6 番起按 50 分结算
