# 拖拉机（升级）规则 — 本实现

固定 4 人，对家为一队（0&2 一队，1&3 一队）。每局按以下阶段推进：

1. 逐张发牌；玩家只能用已经收到的当前级牌亮主。
2. 一张级牌可以亮主，更多张完全相同的级牌可以反主；同样张数不能覆盖已有亮主。
3. 首局最终亮主者成为庄家；以后各局的庄家由上一局胜负决定。无人亮主时保留当前庄位并采用无花色主。
4. 发牌结束后庄家收取底牌，再扣回相同数量的牌，之后开始出牌。

**主牌 = 主花色的所有牌 + 当前级牌的所有花色 + 大小王**。无人亮主时只有当前级牌与大小王是主牌。

首局使用 `first_deal_time` 控制整轮发牌总时间，后续局使用 `deal_time`；首局默认更慢，让玩家有时间抢主。

`removed_rank_count` 用于缩短牌局：`0` 保留全部牌，之后依次移除 `3/4/6/7/8/9/J/Q/A`。例如 `3` 移除 `3/4/6`，`4` 移除 `3/4/6/7`；计分牌 `5/10/K`、`2` 和大小王不会被该设置删除。首局从仍保留的最小级牌开始。

## 一、出牌牌型

一手牌必须是「同组」的下列三种之一：

- **单张**：1 张。
- **对子**：2 张完全相同的牌（同花色同点数，两副牌的同一张算一对）。
  同点数不同花色（如 ♠5+♥5）不是对子，是两张单牌。
- **拖拉机（连对）**：≥2 组花色相同、点数相连的对子（如 ♠3♠3♠4♠4）。
  主牌组内：级牌对彼此不相连；小王对、大王对可与相邻项相连。
  副牌若某点数正好是级牌，会被抽掉，其两侧点数视为相连。

## 二、跟牌规则

- 必须跟出与首家相同的张数。
- 首家出某门时，手上该门（副牌按花色、主牌按主牌组）的牌够则必须全部先跟该门。
- 首家出对子/拖拉机时，手上该门还有对子的必须尽量跟对子。
- 该门不够时可用其他牌补足张数（垫牌 / 毙牌）。

## 三、比大小与得分

- 只有与首家「同牌型」且属主牌或首家副牌花色的牌才参与比大小，主牌永远大于副牌。
- 同组内按点数比大小，最大者赢墩。
- 分值：5 = 5 分，10、K = 10 分。
- 抢到最后一墩者收庄家所扣底牌的分，且底分 × 该墩出牌张数（单张 ×1、对子 ×2、连对 ×N）。

## 四、结算与升级

- 非庄方（闲家队）为进攻方，累计满 blood_start_score（默认 80）即翻庄并升级。
- 未达标则庄方守庄。番数 / 血量按 blood_score_per_unit 折算。
- 级牌沿仍保留的 target_rank 路径推进；到达不高于 final_target_rank 的最高保留级牌后结束。

## 五、AI 概要

- 首攻优先甩连对施压，其次甩低副对子探牌，手牌少时兑现必赢大牌，否则垫最低单张做缺门。
- 跟牌：对家已赢则喂分；对手赢且能赢则出最小可赢牌；副门无分时不轻易用主牌毙，留给后手对家。
